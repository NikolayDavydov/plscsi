SCSI pass through tools exist on and off the web.

Here we offer some screen shots and doc of GUI (graphical user interfaces) and of the default operating system GUI for CLI (command line interfaces).

Surprisingly, in practice some of us actually did find an unsigned Java applet can work as a PLScsi GUI.  Insecurely published code never should have the privilege necessary to pass SCSI thru to an actual device.  But such a program can construct command lines for us to review and forward that we otherwise find hard to remember accurately in detail.

Sorry if you find any of these other tools more difficult to download, make, and run than the PLScsi CLI is.
